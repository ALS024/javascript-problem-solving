# Problem 17 (L-B) ~ Generate Random Number

## Problem

This program generates a random number between minimum and maximum value.

## Solution

```javascript
let min = 0;
let max = 10;

console.log(Math.floor(Math.random() * (max - min + 1)) + min);
```
